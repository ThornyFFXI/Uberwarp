Uberwarp is a plugin used to trigger NPC warp systems via command line.  Configuration files are as follows:

Ashita/resources/ashitahelper/entitydats.xml – Dat location file

-This file is used to store the locations of dats containing entity string tables.  It should not be modified unless you understand what you are doing.

Ashita/resources/ashitahelper/entities/uw***.xml – Entity information files

-These files are used for locating entities the server has failed to load.  They should not be modified unless you understand what you are doing.

Ashita/resources/ashitahelper/uberwarp/***.xml – Warp information files

-These files are used to store information about the various warps that Uberwarp can perform.  You can modify them to change aliases for any system with multiple destinations.  You should not modify any fields other than Alias, though you can copy entire lines if you want multiple aliases for a given destination.

-Modify these files with extreme caution as changes to field other than alias can result in warping to inappropriate locations or sending invalid packets.

 

Note: Configuration files provided with versions of Uberwarp prior to 2.0 or intended for use with Ashita3 will not work correctly.  If you’ve accidentally installed this without removing the earlier files, simply go to Ashita/resources/ashitahelper/entities and delete all files that do not begin with the prefix ‘uw’.

 

As of Ashita4, Uberwarp no longer allows you to automatically override the safety features to use warps you do not have unlocked automatically.  This is due to server-side modifications that make such usage impossible on retail.  If you are on a server that allows this usage, you may remove the bitflag field or change the value to 65536 in the warp information files to remove checks for a given warp.

 

 

Commands

All commands must be prefixed with /uw or /uberwarp.

 

 

/uw hp [required: Alias]

Warp to a home point matching the specified alias.  You must be standing near another home point.

 

 

/uw hp CHECK

List which home points you have not yet unlocked.  You must be standing near a home point.

 

 

/uw wp [required: Alias]

Warp to a waypoint using the adoulin waypoint system.  You must be standing near another waypoint.

 

 

/uw wp CHECK

List which waypoints you have not yet unlocked.  You must be standing near a waypoint.

 

 

/uw sg [required: Alias]

Warp to a survival guide matching the spceified alias.  You must be standing near another survival guide.

 

 

/uw sg CHECK

List which survival guides you have not yet unlocked.  You must be standing near a survival guide.

 

 

/uw aw [required: Zone Name]

Warp to a zone using the abyssea zone system.  Use the name of the zone you're warping to, not the name of the abyssea zone(valkurmdunes not abysseamisareaux).  You must be near an abyssea warp NPC(Horst,Vincent,etc.).

 

 

/uw aw CHECK

List which abyssea zones you have unlocked.  You must be near an abyssea warp NPC.

 

 

/uw uc [required: Zone Name]

Warp to a zone using the unity warp system.  You must be near a unity warp npc.

 

 

/uw uc CHECK

Check which unity warps you have unlocked.  You must be near a unity warp npc.

 

 

/uw pw [required: Zone Name]

Warp to a zone using the proto-waypoint system.  You must be near a city proto-waypoint.

 

 

/uw pw CHECK

Check which proto-waypoint warps you have unlocked.  You must be near a city proto-waypoint.

 

 

/uw ep [required: Target Portal #]

Warp to an eschan portal or ethereal ingress within the same zone.  You must be near a different eschan portal or ethereal ingress.

 

 

/uw ep CHECK

List which eschan portals you have unlocked.  You must be near an eschan portal or ethereal ingress.

 

 

/uw ab [required: Target Conflux #]

Use an abyssea conflux to warp you to another abyssea conflux.  If you have not yet unlocked the starting conflux, it will be unlocked for you(provided you have enough cruor).

 

 

/uw ab CHECK

List which abyssea confluxes you have unlocked.  You must be near an abyssea conflux.

 

 

/uw rp [required: Target Portal]

Use the runic portal in Aht Urhgan Whitegate to warp to another runic portal.  This will not work if you have assault orders.

 

/uw rp CHECK

List which runic portals you have unlocked.  You must be near the runic portal in Aht Urhgan Whitegate.

 

/uw ev [optional: NOWARP]

Get an elvorseal.  If NOWARP is specified, will exit menu afterwards.  If it is not specified, will warp to DI.  You must be near the appropriate NPC, and elvorseal must be available at the time.

 

 

/uw cw

Enter a crystal warp(the blue/purple crystal formations in adoulin that lead between gates zones and ra'kaznar).

 

 

/uw ee

Enter or exit escha from a dimensional portal or undulating confluence.  Also exits altaieu dimensional portal back to crag.

 

 

/uw ea

Enter al'taieu from a crag's dimensional portal.  Use /uw ee to exit from al'taieu to crag.

 

 

/uw sa

Use a scalable area in adoulin.

 

 

/uw cp

Use a castoff point in adoulin.

 

 

/uw cu

Go upstream at a castoff point in foret de hennetiel.  This will do nothing at points that cannot go upstream.

 

 

/uw ae

Enter abyssea using a cavernous maw.

 

 

/uw ad [Optional: Third Argument]

Automatically detect the available warp system, and trigger it's command.  If a third argument is specified, it will be passed to the command.

Notes on /uw ad:

·       It is possible to be in range of 2 systems, such as at northern sandoria HP#1 spaced between the survival guide and home point.  If you are in such a position, there is no guarantee /uw ad will trigger the system you want(/uw ad rabao could hit either the home point or the survival guide).

·       To ensure you get the results you are aiming for, either use the more specific command or ensure you are more than 5.5 yalms away from all but one warp method.

·       Castoff points with 2 options will default to downstream in foret de hennetiel.

·       Dimensional portals will go to Reisenjima, not Al'taieu.

 

Safety/Convenience Features:

·       Uberwarp will automatically coordinate a delay if multiple characters on the same system are trying to use the same NPC at the same time.  You do not need to stagger waits, a simple multisend command is fine.

·       Uberwarp will force the NPC to draw itself if you are close enough to use the NPC and it failed to show up.

·       Uberwarp will verify you have the destination unlocked and enough of the appropriate currency to warp there for every system.

·       Uberwarp completes all menus identically to retail client, for both same zone and other zone warps.

 

Notes on input format:

·       You may skip spaces and punctuation when typing zone names or aliases, but spelling must be correct.

·       Default aliases are zonename, zonename2, zonename3, and so on.  Frontier stations are zonename, bivouacs are zonename#.  Adoulin waypoints are as listed in the configuration files.

·       You may change the aliases for home points, waypoints, or survival guides by going into the xml files in Ashita/resources/ashitahelper/uberwarp/ and changing the 'alias' field.

·       You may use multiple aliases for a single point by copying the entire line and changing the alias for the second copy.

 

Notes on resource files:

·       posx, posy, posz fields refer to the location of the NPC.

·       Landx, Landy, Landz, Landh positions refer to the raw binary values used when warping to those locations from another location in the same zone.

·       Scalable area, castoff point, and 'upstream' castoff point entries store the land values for warping away from that location, rather than to it.  This was done so that only one point would need to be looked up.

 

Notes on ingame warp systems:

·       Talk to Anastase in Ru'lude gardens to unlock proto-waypoints.

·       Complete appropriate mission access to use escha entrances, abyssea entrances.

·       Complete appropriate quests to use scalable areas, castoff points.

·       Complete appropriate cutscenes to use abyssea warps.

·       Kill appropriate wildskeeper reive to use crystal warps.
